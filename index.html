Wie ben ik?

Ik ben Jonas Claes, software ontwikkelaar en student. Ik studeer Toegepaste Informatica aan de Thomas More IT Factory in Geel in het eerste jaar. In mijn vrije tijd ben ik enorm veel bezig met technologie, ik speel met alles van microcontrollers tot netwerken en software. Verder ga ik soms eens met een camera op pad.
Ik ben meer een logische denker, dat wil zeggen dat ik beter in de backend van software thuis ben. Dat wil natuurlijk niet zeggen dat ik niets in de frontend kan maken. Het liefst werk ik samen met collega-ontwikkelaars om zo tot een mooi eindresultaat te komen.
In mijn 6de middelbaar heb ik voor Watertechniek Teunissen in Pelt software mogen bouwen om pompen te testen en hier grafieken van te maken. Dit was zo een success dat ik dit meerdere keren mag doen. Deze software is gebouwd met Node.JS in de backend en het Bootstrap Framework in de frontend. De code van dit project is ook publiek op mijn GitHub, deze kan je hier terug vinden.

Ik ben naast student ook Co‑Founder, CTO en Full Stack Developer bij my-student-id.com sinds Januari 2021.
Verder heb ik mijn eigen smarthome platform ontwikkeld, inclusief hardware, met integratie tussen Amazon Alexa en Google Assistant via een cloud server. In dit platform kunnen ook toestellen van Miele gekoppeld worden via een cloud server, alsook SMA omvormers voor zonnepanelen via Modbus TCP en 3rd partij toestellen door middel van MQTT en/of REST over HTTP. Een smarthome platform zonder app is natuurlijk niet compleet, daarom dat er dus ook een Android app beschikbaar is in de Google Play Store, deze is gemaakt met het Ionic framework.
